02 — MARKS (REFINED)

Three marks. One continuous line.

Every mark passes the Arturo Luz test: reproducible as a single strand of stainless steel. The `t` crossbar is deliberately heavier — the one asymmetric terminal that proves a human hand was here.

PRIMARY WORDMARK · IRON BARK ON ECRU

Quiet elegance. Set in EB Garamond with spacious lowercase letter-spacing (0.12em), achieving ultimate poise, symmetry, and restraint—exactly uniform with the maker's mark.

PRIMARY WORDMARK · ECRU ON OBSIDIAN
LOGOMARK · THE THRESHOLD · IRON BARK + LOKO RUST (SACRED)

Bold lowercase 'k' shape. Styled in the monumentally iconic and creative geometric style of the Cosmos reference poster curation, forming a completely timeless creative anchor.

KTHA MAKER'S MARK

The four-letter signature. Tone-on-tone by default; Loko Rust for sacred editions only. Appears blind-debossed on every print.

08 — THE HANDOFF

Jed builds the shell. You manage within.

Layer 1 · Locked CSS

One file pasted into Code Injection. Restyles all native Squarespace elements. Never edited.

Layer 2 · Snippet Kit

7 Code Block snippets for signature moments: hero, feature-row, quote, wordmark, KTHA stamp, deckled image, calado divider. Edit only marked zones.

Layer 3 · Native Editor

Everything else in the normal Squarespace editor. Galleries, text, spacing, new pages. Layer 1 keeps it on-brand automatically.

Paste Layer 1 once. Build with native sections. Drop Layer 2 for signature moments. Never edit Layer 1.
